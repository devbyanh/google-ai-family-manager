# Google AI Family Manager

Ứng dụng web quản lý đơn hàng và tài khoản bán gói Google AI Pro nhóm gia đình.

## Tính năng
- Quản lý Acc mẹ, phân chia 5 slot tự động.
- Quản lý Đơn hàng, gán vào Acc mẹ.
- Thống kê doanh thu, tỷ lệ lấp đầy.
- Tích hợp Google Sheets làm cơ sở dữ liệu.
- Import dữ liệu từ CSV.
- Backup và khôi phục dữ liệu JSON.

## Cài đặt
1. Khởi chạy bằng bất kỳ web server nào (Live Server, http-server, v.v.)
2. Cấu hình Google Apps Script Web App URL trong phần Cài đặt để đồng bộ dữ liệu.
